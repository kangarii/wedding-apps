
# Wedding Application

This is a wedding application built with HTML, Javascript and Tailwind CSS. It allows users to view and interact with wedding-invite information.

## Features

- Display guest information
- Google Maps
- Wedding event details
- Photo gallery

## Getting Started

1. Clone the repository:

  ```bash
   git clone git@github.com:kangarii/wedding-apps.git
   ```
2. Navigate to the project directory:

  ```bash
  cd wedding-apps
  code . (if u use vs code)
  ```
4. Start the development server:

```
open with live server
```
5. How to change guest name
  ```
  Add "?to=guest_name" 
  example "https://wedding-apps-nine.vercel.app/?to=Your_Firends"
  ```
