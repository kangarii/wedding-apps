
# Wedding Application

This is a wedding application built with HTML, Javascript and Tailwind CSS. It allows users to view and interact with wedding-invite information.

## Features

- Display guest information
- Google Maps
- Wedding event details
- Photo gallery

## Getting Started

1. Clone the repository:

  ```bash
   git clone https://github.com/your-username/wedding-application.git
   ```
2. Navigate to the project directory:

  ```bash
  cd wedding-application
  ```
3. Install the dependencies:

  ```bash
  # If using npm
  npm install

  # If using Yarn
  yarn install
  ```
4. Start the development server:

  ```bash
  # If using npm
  npm run dev

  # If using Yarn
  yarn dev
  ```
5. Open your browser and visit http://localhost:3000 to view the application.
